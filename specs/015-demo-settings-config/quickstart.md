# Quickstart: Demo Settings Config

1. Start Phoenix and Cacablu.
2. Load a Cacablu project with timeline bars.
3. Open `Edit > Demo Settings`.
4. Enter a demo title.
5. Toggle `Loop demo`, `Sound`, and `Debug grid`.
6. Choose a Log detail value.
7. Apply the settings.
8. Verify Phoenix writes `launcher/data/config/control.spo`.
9. Verify `demo_end` matches the latest bar end time in the timeline.
10. Disconnect Phoenix and apply again; verify Cacablu writes an Events entry instead of showing an alert.
